# 角色原型 (2017-10-25)
- 引入Tpye Script
- 玩家: 座標、生命、攻擊，判定動作計時

- 移動: 在世界範圍內任意移動，有動作時間
- 攻擊: 在生命足夠時任意攻擊，怪物死亡判定，有動作時間
